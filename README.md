## 前言

如果要构建项目请先

```bash
go mod tidy
```

这是一个写作的工具集，包含了书籍搜索，书籍下载，批量下载。

## 如何使用？

1. 下载源代码从代码中构建

   ```shell
   git clone https://github.com/aiShuiJiaoDeXioShou/text_processing.git
   go mod tidy
   go build src main.go
   ```
2. 下载windows版本exe文件
